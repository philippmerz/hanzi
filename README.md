# 漢字 Hanzi

A minimalist app for learning Chinese characters. Built with React + TypeScript.

## Features

- **Learn & Test modes** — browse characters freely or quiz yourself with click-to-reveal
- **10 collections** — from basic radicals to philosophy, ~126 characters
- **Dark mode** — true black for AMOLED displays
- **Progress tracking** — learned characters persist in localStorage
- **Keyboard shortcuts** — ← → navigate, Space reveal, M mode, T theme

## Development

```sh
npm install
npm run dev
```

## Deploy

Pushes to `main` auto-deploy to GitHub Pages via the included workflow.
